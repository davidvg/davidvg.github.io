Movimiento
==========

- `Ctrl + f` - Avanzar una pantalla completa
- `Ctrl + b` - Retroceder una pantalla completa
- `Ctrl + d` - Avanzar media pantalla 
- `Ctrl + u` - Retroceder media pantalla
